GOLANG
======

Developed by: 
* Ken Thompson (Unix, B)
* Rob Pike (Plan 9, UTF-8)
* Robert Griesemer (V8 JS, Java Hotspot)

Features:
* Compiled Language
* Functions are First Class Objects
* Strongly typed language
* Go routines and channels 
* Garbage Colections
